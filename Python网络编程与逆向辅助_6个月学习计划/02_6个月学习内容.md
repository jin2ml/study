# Python 网络编程与逆向辅助 6 个月学习内容

---

## 第 1 月：Python 基础与脚本化

### 学习目标
- 熟悉 Python 基本语法
- 掌握常用数据结构
- 能读写文件、处理 JSON、写基础脚本

### 学习内容
- 变量与动态类型
- 字符串、列表、元组、字典、集合
- 条件判断、循环、切片、推导式
- 函数、参数、返回值
- 模块与导入
- 异常处理 `try/except`
- 文件读写
- JSON 读写
- 命令行参数

### 建议练习
- 字符串处理脚本
- JSON 读取与写入脚本
- 文件关键词统计脚本
- 命令行参数解析脚本

### 建议产出
- `hello_python.py`
- `json_reader.py`
- `file_keyword_counter.py`
- `arg_demo.py`

## 第 3 月：HTTP/HTTPS 请求复现

### 学习目标
- 理解 HTTP 请求结构
- 能复现浏览器或抓包中的接口请求
- 能处理登录态和上传下载

### 学习内容
- HTTP 请求行、响应行、状态码
- Header、Cookie、Session、Token
- Query 参数、Form、JSON 请求体
- `requests`
- `httpx`
- 超时、重试、重定向
- 文件上传与下载
- 代理设置

### 建议练习
- GET 请求脚本
- POST JSON 请求脚本
- Session 登录请求脚本
- 自动重试客户端
- 文件上传脚本
- 抓包请求复现脚本

### 建议产出

- `http_post_demo.py`
- `session_login_demo.py`
- `retry_client.py`
