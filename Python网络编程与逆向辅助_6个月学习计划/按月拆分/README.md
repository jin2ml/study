# 按月拆分版学习计划

本目录将 6 个月学习计划继续拆分为按月文档，便于逐月执行。

## 文件结构

- `第1月_学习内容.md`
- `第1月_学习测验.md`
- `第2月_学习内容.md`
- `第2月_学习测验.md`
- `第3月_学习内容.md`
- `第3月_学习测验.md`
- `第4月_学习内容.md`
- `第4月_学习测验.md`
- `第5月_学习内容.md`
- `第5月_学习测验.md`
- `第6月_学习内容.md`
- `第6月_学习测验.md`

## 使用建议

1. 月初先看当月 `学习内容`
2. 按周推进学习与练习
3. 月末完成当月 `学习测验`
4. 将脚本、笔记、复盘整理到同月目录下
