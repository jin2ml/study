# 第 2 月学习测验：数据处理、编码解码与哈希签名

## 理论题
1. `bytes` 和 `str` 的区别是什么？
2. Base64、Hex、URL 编码分别常见于什么场景？
3. HMAC 和普通哈希的区别是什么？
4. 为什么有些响应内容需要先解压再解析？

## 实操题
1. 写一个支持 Base64/Hex/URL 的编码解码工具。
2. 写一个脚本，输入消息和密钥输出 HMAC-SHA256。
3. 写一个脚本，对 gzip 或 zlib 数据进行解压。
4. 从 JSON 响应中提取指定路径字段。

## 通过标准
- 能独立完成 4 道实操题中的 3 道以上
- 能解释常见签名和编码问题
- 能快速写出数据转换工具
