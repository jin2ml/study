# 第 4 月学习内容：Socket、TCP/UDP 与简单协议

## 本月目标
- 理解网络通信的字节流本质
- 掌握 TCP/UDP 基础编程
- 能打包解包简单二进制协议

## 每周安排

### 第 1 周：Socket 基础
- `socket` 模块
- TCP 与 UDP 区别
- 客户端与服务端基础概念

### 第 2 周：TCP/UDP 实操
- TCP client
- TCP server
- UDP 发送与接收

### 第 3 周：字节流与 struct
- 字节流处理
- `struct.pack()` / `struct.unpack()`
- 长度字段、类型字段、载荷字段

### 第 4 周：简单协议实验
- 设计小协议
- 粘包/拆包基础思路
- 报文解析

## 建议练习
1. 写一个 TCP 客户端
2. 写一个 TCP 服务端
3. 写一个 UDP 发送接收程序
4. 写一个包含“长度 + 类型 + 数据”的报文打包解包脚本

## 建议产出
- `tcp_client.py`
- `tcp_server.py`
- `udp_demo.py`
- `packet_struct_demo.py`
- `simple_protocol_demo.py`

## 本月完成标准
- 能写基础 TCP/UDP 脚本
- 能理解简单二进制报文结构
- 能解释粘包/拆包的基础原因
