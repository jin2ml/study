# 第 1 月学习测验：Python 基础与脚本化

## 理论题
1. `list` 和 `tuple` 的主要区别是什么？
2. `dict.get()` 和 `dict[key]` 有什么差异？
3. `try/except/finally` 各自的作用是什么？
4. 为什么很多脚本会写 `if __name__ == '__main__':`？

## 实操题
1. 编写脚本，统计文本文件中每个单词出现次数。
2. 编写脚本，读取 JSON 文件并提取指定字段。
3. 编写脚本，接收命令行参数并执行不同逻辑。

## 通过标准
- 能独立完成 3 道实操题中的 2 道以上
- 能熟练操作文件、JSON、列表、字典
- 能解释基础语法和模块结构
