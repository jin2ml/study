# 第 1 月学习内容：Python 基础与脚本化

## 本月目标
- 完成 Python 语法迁移
- 能独立写基础脚本
- 能处理文件、JSON、命令行参数

## 每周安排

### 第 1 周：语法与数据结构
- 变量与动态类型
- 字符串、列表、元组、字典、集合
- `if / for / while`
- 切片与推导式

### 第 2 周：函数与模块
- 函数定义与返回值
- 默认参数、关键字参数
- 模块导入
- 常用内置函数
- `if __name__ == '__main__'`

### 第 3 周：文件与异常
- 文本文件读写
- JSON 文件读写
- `try/except/finally`
- 路径处理
- 基础日志输出

### 第 4 周：脚本化练习
- 命令行参数解析
- 目录遍历
- 关键词统计
- 批量处理文本文件

## 建议练习
1. 写一个字符串清洗脚本
2. 写一个 JSON 读取与格式化脚本
3. 写一个统计文件关键词出现次数的脚本
4. 写一个命令行参数处理脚本

## 建议产出
- `hello_python.py`
- `json_reader.py`
- `file_keyword_counter.py`
- `arg_demo.py`

## 本月完成标准
- 能独立写 100 行以内小脚本
- 熟练使用 `str`、`list`、`dict`
- 能完成文件读写和异常处理
