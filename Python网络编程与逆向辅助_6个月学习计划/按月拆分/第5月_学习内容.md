# 第 5 月学习内容：asyncio、并发请求与 WebSocket

## 本月目标
- 掌握协程与异步请求基础
- 能写批量请求工具
- 能连接 WebSocket 并观察消息流

## 每周安排

### 第 1 周：asyncio 基础
- 协程
- 事件循环
- `await`
- 并发任务调度

### 第 2 周：异步 HTTP 请求
- `httpx.AsyncClient` / `aiohttp`
- 并发抓取
- 超时控制
- 重试机制

### 第 3 周：并发工具化
- 并发数限制
- 限速控制
- 日志输出
- 结果汇总

### 第 4 周：WebSocket
- WebSocket 握手
- 消息收发
- 心跳包
- 基础消息处理

## 建议练习
1. 写一个异步批量请求器
2. 写一个并发下载器
3. 写一个带日志、超时、重试的异步客户端
4. 写一个 WebSocket 客户端

## 建议产出
- `async_fetcher.py`
- `batch_downloader.py`
- `async_retry_client.py`
- `ws_client_demo.py`

## 本月完成标准
- 能写异步请求工具
- 能控制并发、超时、重试
- 能连接 WebSocket 并处理消息
