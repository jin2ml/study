# 第 3 月学习内容：HTTP/HTTPS 请求复现

## 本月目标
- 理解 HTTP 请求结构
- 能用 Python 复现常见接口
- 能处理 Cookie、Session、Token、上传下载

## 每周安排

### 第 1 周：HTTP 基础
- 请求行、请求头、请求体
- 状态码
- Query 参数与 Body
- 常见 Content-Type

### 第 2 周：requests / httpx 基础
- GET / POST 请求
- Header 设置
- JSON 提交
- form 提交

### 第 3 周：登录态与会话
- Cookie
- Session
- Token
- 重定向、超时、重试

### 第 4 周：请求复现与上传下载
- 文件上传
- 文件下载
- 代理设置
- 按抓包记录复现请求

## 建议练习
1. 写一个 GET 请求脚本
2. 写一个 POST JSON 请求脚本
3. 写一个带 Session 的登录脚本
4. 写一个文件上传脚本
5. 根据抓包信息复现接口请求

## 建议产出
- `http_get_demo.py`
- `http_post_demo.py`
- `session_login_demo.py`
- `retry_client.py`
- `upload_demo.py`
- `replay_request.py`

## 本月完成标准
- 能复现普通 HTTP/HTTPS 接口
- 能处理 Header、Cookie、Session
- 能定位常见请求失败原因
