# 第 2 月学习内容：数据处理、编码解码与哈希签名

## 本月目标
- 掌握常见编码解码方式
- 熟悉 `bytes` 与 `str` 的转换
- 能完成常见哈希、HMAC、压缩处理

## 每周安排

### 第 1 周：JSON 与正则
- `json.loads()` / `json.dumps()`
- 正则匹配、分组、替换
- JSON 字段提取

### 第 2 周：编码与字节处理
- Base64
- Hex
- URL 编码
- `bytes` / `str` 转换

### 第 3 周：哈希与 HMAC
- `md5`
- `sha1` / `sha256`
- `hmac`
- 时间戳与签名拼接思路

### 第 4 周：压缩与解压
- `gzip`
- `zlib`
- `binascii`
- 响应解压验证

## 建议练习
1. 写一个 Base64/Hex/URL 编码解码工具
2. 写一个 MD5/SHA256/HMAC 工具
3. 写一个 gzip/zlib 压缩解压脚本
4. 写一个 JSON 字段提取脚本

## 建议产出
- `encode_decode_tool.py`
- `hash_tool.py`
- `gzip_tool.py`
- `json_extract_tool.py`

## 本月完成标准
- 能区分文本、字节、编码串、压缩内容
- 能独立写编码解码和哈希脚本
- 能处理常见接口返回中的压缩与字节内容
