# Python 网络编程与逆向辅助 6 个月学习计划

本文档集合将内容拆分为 3 类，避免路线、内容、测验混在一起。

## 文档目录

1. `01_6个月学习路线.md`
   - 适合先整体查看半年安排
   - 重点看阶段目标、月份规划、节奏安排

2. `02_6个月学习内容.md`
   - 适合按月学习时查阅
   - 重点看每月知识点、练习内容、项目产出

3. `03_6个月学习测验.md`
   - 适合每月复盘和阶段自测
   - 重点看理论题、实操题、通过标准、总结业测验

## 使用建议

- 第一步：先看 `01_6个月学习路线.md`
- 第二步：每个月按 `02_6个月学习内容.md` 执行
- 第三步：月末做 `03_6个月学习测验.md`
